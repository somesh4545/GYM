# GYM
Easily manage your gym with this app. 

Features:
<ul>
  <li>Add members, add services and plans to them</li>
  <li>Add services</li>
  <li>Add plans</li>
  <li>Monitor members plan expiry</li>
  <li>Detailed collection report</li>
   <li>Cost effective</li>
</ul>
